Brief Introduction to ECMA6

ECMA6 is the newest update to the JavaScript spec. It introduces a lot of new and interesting features and updates that can and will look foreign to you if you've never seen it before.

More importantly though, a lot of the newer plugins and libraries that are being developed are written entirely in JS6 (including ReactJS).

We will now briefly cover a few of the most imporant aspects of JS6 including:

  1. Imports and exports
  2. fat arrow functions and their implications
  3. Some useful new object and array methods

ALSO: what the heck was up with all that gulp stuff? We will discuss why we need a transpiler when using ecma6.

Object Oriented Implementation

Requirements

Ideally, we now have a functioning image slider. Let's not define what an "object" version of this should look like:

JavaScript

var s = new Slider({
    // options
});

s.goToSlide(1); // takes you to slide one
s.pause(); // pauses slide
// ... etc ...

More importantly, we should be able to do the following:

var s = new Slider({
    // options
});
var t = new Slider({
    // options
});

s.goToSlide(1); // takes you to slide one
t.goToSlide(4); // takes you to 4 for the _other_ slide
s.pause(); // pauses slide
t.start(); // starts slide on t
// ... etc ...
